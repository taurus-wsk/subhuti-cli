# Mac 通用技能库

## 系统操作
- open_app: 打开 Mac 应用
- close_app: 关闭指定应用
- hide_app: 隐藏当前活动窗口
- lock: 锁定屏幕
- restart: 重启电脑

## 文件操作
- open_desktop: 打开桌面文件夹
- open_downloads: 打开下载文件夹
- open_folder: 打开指定文件夹
- screenshot: 截图并保存

## 输入操作
- type_text: 输入文字
- click: 鼠标点击指定坐标
- scroll_up: 向上滚动
- scroll_down: 向下滚动

## 网络操作
- wifi_on: 打开 Wi-Fi
- wifi_off: 关闭 Wi-Fi

## 声音操作
- volume_up: 音量增加 10%
- volume_down: 音量减少 10%
- mute: 静音

## 其他
- sleep: 等待指定秒数
- open_url: 打开网页 URL
